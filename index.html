Swipe Side Navigations

Side menus became popular when developers started using it in their mobile apps where there is no much space on the screen. It is natural for any device right now, and can be also intuitive. And it will look amazing for a desktop version of your site too.

Smooth Animations

You can see that animations look really smooth. It is provided by the Khaki core, and it is also included in this template. Slidings, counters, buttons animations, a parallax, appearing and disappearing - these features are some kind of a language your site can talk.
